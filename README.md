# Engenharia_de_Prompt_Aplicacoes_Ai
Atividades_da_Faculdade
Aplicações da Engenharia de Prompt
Chatbots de atendimento:
Prompts bem estruturados ajudam o chatbot a entender melhor a intenção do usuário e fornecer respostas rápidas e corretas.

Assistente virtual:
Prompts claros permite que o assistente execute tarefas como agendar reuniões, criar lembretes ou resumir textos.

Geração de conteúdo:
Ao especificar o formato, tom e objetivo do texto, é possível gerar artigos, específicos de produtos ou postagens para redes sociais com mais eficiência.

Análise de dados:
Prompts podem ser usados ​​para pedir resumos, insights ou visualizações de dados em linguagem natural.
